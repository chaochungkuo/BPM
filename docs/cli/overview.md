# CLI Overview

Top-level commands:

- `bpm project` ‚Äî create and inspect BPM projects.
- `bpm resource` ‚Äî manage BRS stores (add/activate/list/info/update/remove).
- `bpm template` ‚Äî render/run/publish templates from the active BRS.
- `bpm workflow` ‚Äî run workflows (if present in a BRS).
- `bpm agent` ‚Äî provider-backed chat assistant for BPM/BRS guidance (`config`, `doctor`, `start`, `history`).

Use `--help` on any command for details.
