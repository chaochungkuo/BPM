# BRS Anatomy

Key points:
- Recommended layout: `repo.yaml`, `config/`, `templates/`, `workflows/`, `hooks/`, `resolvers/`.
- Discovery: BPM scans for `templates/*/template_config.yaml` and `workflows/*/workflow.yaml`.
- Metadata: `repo.yaml` names the store and sets compatibility/version info.
